Вывод данных при откладке


## Установка (composer)
```sh
composer require 'rusadrako/debug'
```


## Установка (manual)
- Скачать и распоковать библиотеку.
- Добавить в код инструкцию:
```php
require_once('/debug/src/autoload.php')
```
